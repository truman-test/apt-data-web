# MVP 요구사항 명세서

**작성일**: 2026-01-14
**버전**: 1.0
**상태**: 확정됨

---

## 1. 프로젝트 개요

### 1.1 목표
전국 아파트 실거래가 데이터 기반의 부동산 정보 서비스 MVP 구축

### 1.2 데이터 소스
- apt-data-collector-v2에서 수집한 PostgreSQL 데이터
- 매매 실거래가: 10,602,206건
- 전월세 실거래가: 11,844,959건
- 단지 마스터: 44,571건
- 학교 정보: 12,610건
- 지하철역 정보: 1,090건

---

## 2. 기능 요구사항

### 2.1 MVP 페이지 구성 (5개)

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 홈 | `/` | 검색 UI, 주요 기능 소개 |
| 검색 결과 | `/search` | 검색 결과 목록 표시 |
| 단지 상세 | `/apt/[id]` | 단지 정보 + 시세 차트 |
| 지도 | `/map` | 네이버 지도 기반 탐색 |
| 비교 | `/compare` | 복수 단지 시세 비교 |

### 2.2 검색 기능

| 항목 | 내용 |
|------|------|
| 검색 대상 | 아파트명, 지역명 |
| 지역 필터 | 시/도 → 구/군 계층형 드롭다운 |
| 결과 표시 | 단지명, 주소, 세대수, 최근 시세 |

### 2.3 지도 기능

| 항목 | 내용 |
|------|------|
| 지도 API | 네이버 지도 |
| 마커 표시 | 단순 마커 (클러스터링 없음) |
| 마커 정보 | 단지명 + 평당가 |
| 지역 탐색 | 드롭다운 선택 + 지도 이동/줌 병행 |

### 2.4 단지 상세 정보

| 섹션 | 표시 정보 |
|------|----------|
| 기본 정보 | 단지명, 주소, 세대수, 동수, 준공년도 |
| 평형 정보 | 전용면적, 공급면적, 해당 세대수 |
| 매매 시세 | 최근 거래가, 가격 추이 차트 |
| 전월세 시세 | 전세가, 월세 보증금/월세 |
| 교통 | 최근접 지하철역, 거리, 도보시간 |
| 학군 | 주변 학교 (초/중/고), 거리 |

### 2.5 시세 차트

| 항목 | 내용 |
|------|------|
| 차트 유형 | 월별 평균가 라인 차트 + 개별 거래 산점도 |
| 필터링 | 평형(전용면적)별 필터링 필수 |
| 기간 선택 | 1년 / 3년 / 5년 / 전체 |
| 거래 유형 | 매매 / 전세 / 월세 탭 구분 |

### 2.6 비교 기능

| 항목 | 내용 |
|------|------|
| 비교 대상 | 최대 3~4개 단지 |
| 비교 항목 | 시세 추이, 평당가, 기본 정보 |

## 3. 비기능 요구사항

### 3.1 반응형 디자인
- 모바일 반응형 필수 (Mobile First 권장)
- Breakpoints: Mobile (< 768px), Tablet, Desktop

### 3.2 성능
- 검색 응답: < 500ms
- 페이지 로드: < 2s (LCP)
- 지도 마커 렌더링: 1000개 이하 동시 표시

### 3.3 기술 스택
- Frontend: Next.js 14+ (App Router)
- 지도: Naver Maps API
- 상태관리: React Query + Zustand
- 차트: Recharts
- 스타일: Tailwind CSS

## 5. 사용자 플로우

### 5.1 검색 플로우
```
홈 → 검색어 입력 → 검색 결과 → 단지 선택 → 상세 페이지
```

### 5.2 지도 탐색 플로우
```
지도 페이지 → 지역 선택/지도 이동 → 마커 클릭 → 상세 페이지
```

### 5.3 비교 플로우
```
상세 페이지 → "비교하기" 선택 → 비교 페이지 → 단지 추가
```
